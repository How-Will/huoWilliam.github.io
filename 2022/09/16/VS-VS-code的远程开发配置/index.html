  2. 安装远程开发插件：在插件商店搜索 Remote Development，并安装。

  3. 配置远程服务器

    • 使用快捷键 ctrl + shift + p，打开命令面板，搜索命令 “Open SSH Configuration File”，然后打开弹出的配置文件 C:\Users\huo\.ssh\config

    • 在配置文件中，配置 Host，Hostname，User，具体配置如下：

  4. 修改设置：使用快捷键 ctrl+,，打开设置面板，搜索设置 ”Always reveal the SSH login terminal“，将其选中打开

  5. 使用快捷键 ctrl + shift + p，打开命令面板，搜索命令 “Connect Current Window to Host”，便可以使用 SSH 连接到服务器了。接下来的编程操作便都是在服务器上进行的了。
